# Assets Directory

This directory contains all static assets for the Silent Moon meditation app.

## Structure

- `/images` - Illustrations, photos, and image assets
- `/icons` - App icons and UI icons
- `/audio` - Meditation audio files, sleep stories, and ambient sounds

## Image Assets

### Current Images
- `signup-illustration.png` - Welcome screen illustration

### Required Images
- App icon (1024x1024)
- Splash screen
- Adaptive icon (Android)
- Favicon (Web)
- Meditation category icons
- Course thumbnails
- Profile placeholders

## Audio Assets

### Categories
- Guided meditations
- Sleep stories
- Ambient sounds
- Nature sounds
- Bedtime music

## Icon Assets

### UI Icons
Using @expo/vector-icons (Ionicons) for most UI elements

### Custom Icons
- App logo/moon icon
- Category icons
- Achievement badges
